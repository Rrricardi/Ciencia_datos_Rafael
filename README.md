## 👋 Sobre Este Repositorio

¡Hola! 👋 Bienvenido a mi espacio de aprendizaje práctico. Este repositorio contiene una serie de **notebooks interactivos** que documentan mi progreso en el dominio de Python para Ciencia de Datos.

Cada proyecto está diseñado para resolver un problema específico, desde la manipulación básica de datos hasta la visualización avanzada y el análisis numérico. Mi objetivo es **aprender haciendo**, y este repositorio es mi portafolio vivo de ese journey.

> 💡 *"La teoría sin práctica es estéril. La práctica sin teoría es ciega."*
